Упаковщик данных для ZX Spectrum "Hrum 3.5i", PC-версия

                     version 1.1b



Данная программа - переделанная версия упаковщика с ZX Spectrum. Автор
оригинальной программы - Hrumer, автор PC-версии - Nikita Burnashev,
подключение депакера добавил psb.

Файлы:

hrum35.exe	- упаковщик
example.bat	- пример использования упаковщика в bat-файлах



Программа поддерживает следующие ключи:

-opt		оптимизация, практически не влияет на степень сжатия (выкл. по умолчанию)
-ei		разрешить прерывания по выходу из депакера (да по умолчанию)
-di		запретить прерывания по выходу из депакера (нет по умолчанию)
-start n 	стартовый адрес блока (с депакером) (#6000 по умолчанию)
-depackto n 	адрес распаковки (#6000 по умолчанию)
-dadr n		адрес депакера (#5B89 по умолчанию)
-sp n		новое значение регистра SP после распаковки (нет по умолчанию)
-jp n		переход по адресу после распаковки (нет по умолчанию)
